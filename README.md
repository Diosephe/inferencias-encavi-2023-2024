# ABP 5 – Inferencia Estadística (ENCAVI 2023–2024)

Proyecto del Módulo 4: Inferencia Estadística usando ENCAVI 2023–2024 (Chile).
Se trabaja con una base reducida (jóvenes 18–29) y se incluye un dataset simulado (n=100) como entregable.

## Contenido del repositorio
- `informe/`: informe en LaTeX y PDF final.
- `data/`:
  - `encavi_jovenes_18_29.csv` (n=2054, 9 variables)
  - `dataset_simulado_100.csv` (n=100, 9 variables)
- `figures/`: gráficos TLC usados en el informe.
- `slides/`: presentación Beamer (fuente `.tex`).

## Cómo compilar
### Informe
Compila `informe/informe.tex` (Overleaf o PDFLaTeX).

### Slides (Beamer)
Compila `slides/presentacion_beamer.tex`.
El archivo asume que las figuras están en `figures/`.

## Fuente de datos
ENCAVI 2023–2024 (Chile), datos abiertos (datos.gob.cl).
